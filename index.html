What is WebCatalog?

WebCatalog is a desktop app for macOS, Windows & Linux which allows you to install and run any web apps natively.

It works similarly to Fluid and Nativefier, but offers more features and an user-friendly user interface. Don't trust me? Check the Comparison table.
